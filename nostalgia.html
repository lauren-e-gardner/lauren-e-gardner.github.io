Nostalgia



N O S T A L G I A


Nostalgia is a common driving force in my work; I am often challenged with these feelings when I look at childhood photos, play games from my youth, or even see certain color palettes. My goal with this project was to explore these topics and combine them in a way that expressed my nostalgic feelings.




My journey as an artist began on a Windows 7 computer -- specifically on MS paint. As a young girl I would use my mouse to draw pixel art on the application. I revisited this platform and attempted to use it as I would when I was younger. I started with a single photograph - a childhood photo taken by my mom - and drew it using my mouse and the default color palette.




While I drew, I considered the relation between the childhood photo I chose and the program I drew in. They both made me feel a specific type of nostalgia. I came to see my younger self as a snapshot in time - like an older version of a computer program that still exists. Much like MS paint, I have many versions. Each one builds on the last, adding new features and connections. Often times, the old is left behind and forgotten in favor of the new. So I decided to keep looking at the old, and I explored all of the versions of MS paint that I could find.


During my exploration I found something interesting, the Windows 3.0 Paint v.2 was only able to represent colors using a 3-bit representation: White, Black, Red, Green, Blue, Yellow, Green/Blue, and Purple in their purest forms. All other colors are some combination of these 8 pixels. I found these dithering patterns interesting and even coded my own dithering algorithm to recreate this style.


By reflecting on each version of the program, I came to understand the improvements made to each version - like the addition of new colors. Furthermore, to better understand the programs initial layout and design, I decided to try drawing in a physical medium from my childhood - crayon.


At this point I felt like I understood the choices made when designing the UI/UX of MS paint as well as the updates and improvements made with each version. In my final iteration of this project I wanted to combine my findings into a single piece. I used colored filters and a printer to overlay each drawing on top of each other to show the ways in which all the versions are related.


Finally, I decided to import the default MS paint palette into my current drawing program - procreate. This way, I can use an new interface while still keeping some familiarity from the first drawing program I ever used.


This project helped me to realize the importance of exploring the past to improve the future and how this point not only relates to a computer program but also to someones life. These version are what makes a person who they are - the old and the new are always intertwined - often with the new making improvments based on the old.
